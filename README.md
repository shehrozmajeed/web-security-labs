# Web Security Labs

Writeups from PortSwigger Web Security Academy — hands-on labs covering 
real-world web vulnerabilities. Each folder is a vulnerability category. 
Each file documents my methodology, payloads used, and key takeaways.

## About
These are not copy-pasted solutions. Every writeup reflects my own 
approach — what I tried, what failed, and what worked. Written to 
reinforce learning and demonstrate practical web security skills.
